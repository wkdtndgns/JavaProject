static 메소드와 instance 메소드의 차이는 다음과 같다.
* instance 메소드에서는 this 키워드를 사용할 수 있고, 
* static 메소드에서는 this 키워드를 사용할 수 없다.

* instance 메소드를 호출할 때에는 this 객체를 명시해 주어야 한다.
* static 메소드를 호출할 때에는 this 객체를 명시하지 않아도 된다

부모 클래스 ( 상속 가능 ) 
1 공통 메소드 상속  (코드 중복제거) 
		      ( 코드 재사용 ) 
2 다형성 구현  ( 호환되는 클래스들   
       그 클래스들을 사용하는  코드의 중복 제거 )
